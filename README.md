Instalação
~~~~~~~~~~

instalação rápida:

    git clone https://github.com/ylgnerbecton/bexs-challenge.git
    cd desafio-synergia
    python3 -m venv .env
    source .env/bin/activate
    pip3 install -r requirements.txt
    python3 manage.py runserver
